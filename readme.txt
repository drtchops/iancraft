OPTIONS:
    Go to C:/Users/<you>/Documents/My Games/iancraft for Windows
    Or ~/.local/share/iancraft for *nix
    open user.ini

    options are:
        fullscreen: 0 for windowed mode, 1 for fullscreen, 2 for windowed fullscreen
        window_width/window_height: resolution when windowed
        fullscreen_width/fullscreen_height: resolution when fullscreen
        scroll_width: how large in % the edge is for scrolling the camera with the mouse
        camera_speed: speed of camera in pixels/second as if you were at 1080p
        show_fps: show the approximate fps in the top left
        fps_interval: time between fps updates in ms

    defaults are in defaults.ini
